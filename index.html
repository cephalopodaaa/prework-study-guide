HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Use the 'a' tag to add a link

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Linear and Radial gradients can be used to make cool colour effects!
  • 'card' class can be used to put a box around content

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • changes to a branch can be made by first using the 'add' command, followed by the commit command.
  • changes are only confirmed when the branch is 'pushed' to github. The merge must then be accepted on the github website or desktop app
  • A guide about deploying
  • Here is some guides on github pages and how to set it up.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • the basics of javascript: https://developer.mozilla.org/en-US/docs/Learn/Getting_started_with_the_web/JavaScript_basics

  • final stateement: how it iterates over each item. e.g. x++ which is the same as x+1
